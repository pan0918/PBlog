## 搭建过程

搭建这个博客花了不少时间，但过程很有趣。

### 遇到的挑战

1. Tailwind CSS v4 的配置方式和 v3 有很大不同
2. Next.js 16 的 App Router 需要理解 Server/Client 组件的边界
3. 毛玻璃效果在不同浏览器上的兼容性

### 收获

- 更深入地理解了 React Server Components
- 学会了如何用 Framer Motion 实现流畅的动画
- 掌握了 Markdown 渲染管线的配置

总的来说，这是一次很有价值的实践！
