## 欢迎来到我的博客！

这是我使用 **Next.js + React + Tailwind CSS** 搭建的个人博客网站。

### 技术栈

- **框架**: Next.js 16 (App Router)
- **UI**: React 19 + Tailwind CSS v4
- **动画**: Framer Motion
- **Markdown**: unified + remark + rehype
